# KNN Decision Boundary Visualization

## Description

This project demonstrates the **K-Nearest Neighbors (KNN)** classification algorithm by generating a synthetic dataset and visualizing the decision boundaries for different values of **K (1, 2, and 5)**. The project helps illustrate how the choice of K affects the classifier's decision regions.

---

## Features

- Generate a synthetic dataset using NumPy
- Create binary class labels
- Visualize the dataset using scatter plots
- Train KNN classifiers with different values of K
- Plot decision boundaries for each classifier
- Compare the effect of different K values on classification
